Deraining

雨滴を除去するAI技術で、雨天屋外での撮影の視認性をアップ

Deraining

カメラレンズ表面についた雨などの水滴を、撮影後の画像からAIにより除去する技術です。車載カメラや監視カメラなど雨天屋外での撮影の視認性を向上します。

スライダーを左右に動かすと処理前後の画像が確認できます。

USE CASE

車載カメラ（バックモニター、デジタルミラーの雨天時被写体認識精度向上）

事故や衝突などの緊急事態が発生した際に、撮影後の画像から雨滴や水滴を除去することで、事故の状況や被害の詳細を明確に捉えることができます。事故原因の特定や保険請求の際に役立ちます。

屋外用ロボット（清掃用ロボット、搬送用ロボットの前方認識精度向上）

雨天下での災害発生時などにロボットが撮影した画像から雨滴を取り除くことで、状況をより正確に素早く把握することができます。

屋外監視カメラ（雨天時の監視精度向上）

被写体や人物の識別が容易になり、事故などの抑止や安全確保に貢献します。雨天下でもクリアな映像が入手できるため、分析などの作業効率も向上します。

ドローン （雨天時撮影の画像鮮明化）

災害調査や土地測量など、視覚情報が必要な様々な分野で、雨天時でも正確なデータ収集と分析が可能となります。

APPLICATIONS

IMAGE PROCESSING

  • Image Noise Reduction

    暗所撮影などで発生する高感度ノイズをAIにより低減し高画質化する技術

  • Video Noise Reduction

    長秒露光が難しい動画撮影でもリアルタイムでAIによる高画質化を実現

  • Super Resolution

    低解像度のデジタル画像を、AIにより鮮明さを保持したまま高解像化

  • Deblurring

    手ブレ、動く被写体の撮影により発生したブレをAIにより低減

  • Deraining

    カメラレンズ表面についた水滴を撮影後の画像からAIにより除去

IMAGE RECOGNITION

  • Object Detection

    人や車両などが画像中のどこにあるかを検出する技術

  • Segmentation

    物体の領域を塗り分ける技術

  • Pose Estimation

    人間の関節や主要点（首・肩・肘・手首・足首など）の位置とそれらの接続関係を推定

  • Classification

    画像に写る物体を認識、何が写っているかを出力

  • Anomaly Detection

    正常画像のみの学習により異常箇所を検知
