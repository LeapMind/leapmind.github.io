Classification

画像に写っているものを認識し、何らかの特徴やルールに基づいてどのカテゴリに属するか分類する技術です。画像分類とも呼ばれます。

USE CASE

商品分類

スーパーマーケットやECサイトなどで商品の識別に活用できます。例えば、カメラで商品を撮影し、モデルがそれを検出・分類して正確な商品情報や価格を提供することができます。

APPLICATIONS

IMAGE PROCESSING

  • Image Noise Reduction

    暗所撮影などで発生する高感度ノイズをAIにより低減し高画質化する技術

  • Video Noise Reduction

    長秒露光が難しい動画撮影でもリアルタイムでAIによる高画質化を実現

  • Super Resolution

    低解像度のデジタル画像を、AIにより鮮明さを保持したまま高解像化

  • Deblurring

    手ブレ、動く被写体の撮影により発生したブレをAIにより低減

  • Deraining

    カメラレンズ表面についた水滴を撮影後の画像からAIにより除去

IMAGE RECOGNITION

  • Object Detection

    人や車両などが画像中のどこにあるかを検出する技術

  • Segmentation

    物体の領域を塗り分ける技術

  • Pose Estimation

    人間の関節や主要点（首・肩・肘・手首・足首など）の位置とそれらの接続関係を推定

  • Classification

    画像に写る物体を認識、何が写っているかを出力

  • Anomaly Detection

    正常画像のみの学習により異常箇所を検知
