Video Noise Reduction

動画の高画質化をリアルタイムに実現

暗所撮影などで発生する高感度ノイズをAIにより低減し高画質化する技術です。
長秒露光が難しい動画撮影でも、エッジデバイス上でリアルタイムにAIによる高画質化を実現できます。

この動画には音声はありません。

USE CASE

スマートフォン （ナイトモード動画撮影　ー　低照度高画質撮影）

夜のキャンプや夜景など低照度下での撮影、コンサートやライブイベント、結婚式などの高ISO設定での撮影、スポーツ観戦やペットなど動きのあるシーン、レストランやカフェなど屋内での撮影に効果を発揮します。

中継用カメラ （夜間や暗所での災害・事件のライブ映像の高画質化）

夜間や暗所での撮影時に生じるノイズを低減し、高画質な映像をリアルタイムに配信できるため、正確で迅速な情報伝達が必要とされる災害や事件の報道などで特に効果を発揮します。

アクションカメラ（ 夜間撮影）

サッカーや野球など夜間の屋外スポーツでは選手の動きをより鮮明に捉えます。キャンプや釣り、天体観測などのアウトドアアクティビティでもノイズや画質の劣化を抑え、夜間の瞬間を美しく捉えることができます。

ドローン （夜間、暗所撮影時の高画質化）

高画質な映像と機動性を組み合わせることで、夜間や暗所での撮影において優れたパフォーマンスを発揮します。建築物やインフラの点検、災害状況の把握と救助活動支援、映画や広告制作、イベントやコンサートの撮影などに活用できます。

監視カメラ （夜間監視　－　災害・自然・港湾・セキュリティ）

夜間や暗所での映像品質が向上し、安全性とセキュリティの向上に貢献します。災害発生時の被害状況記録や、救助活動の支援、船舶や不審行動の監視による港湾の安全確保、建物などへの不正侵入や異常活動の早期発見ができます。

APPLICATIONS

IMAGE PROCESSING

  • Image Noise Reduction

    暗所撮影などで発生する高感度ノイズをAIにより低減し高画質化する技術

  • Video Noise Reduction

    長秒露光が難しい動画撮影でもリアルタイムでAIによる高画質化を実現

  • Super Resolution

    低解像度のデジタル画像を、AIにより鮮明さを保持したまま高解像化

  • Deblurring

    手ブレ、動く被写体の撮影により発生したブレをAIにより低減

  • Deraining

    カメラレンズ表面についた水滴を撮影後の画像からAIにより除去

IMAGE RECOGNITION

  • Object Detection

    人や車両などが画像中のどこにあるかを検出する技術

  • Segmentation

    物体の領域を塗り分ける技術

  • Pose Estimation

    人間の関節や主要点（首・肩・肘・手首・足首など）の位置とそれらの接続関係を推定

  • Classification

    画像に写る物体を認識、何が写っているかを出力

  • Anomaly Detection

    正常画像のみの学習により異常箇所を検知
